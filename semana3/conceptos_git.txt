Repositorio:
Un repositorio es el espacio donde se almacenan todos los archivos de un proyecto junto con su historial de cambios. Permite organizar el trabajo y mantener un control de las diferentes versiones del proyecto a lo largo del tiempo.

Commit:
Un commit es un registro que guarda los cambios realizados en los archivos de un proyecto en un momento específico. Cada commit incluye un mensaje descriptivo que ayuda a identificar qué modificaciones se realizaron.

Push:
Push es la acción de enviar los cambios guardados localmente en Git hacia un repositorio remoto, como GitHub. Esto permite compartir el trabajo con otros desarrolladores y mantener una copia actualizada del proyecto en línea.
