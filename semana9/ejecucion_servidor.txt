Cuando se ejecuta el comando node server.js, Node.js inicia la aplicación y el servidor comienza a escuchar solicitudes en el puerto 3000. En la terminal aparece el mensaje "Servidor ejecutándose en puerto 3000", indicando que el servidor se inició correctamente. Al visitar la ruta principal (http://localhost:3000), el navegador muestra el mensaje "Servidor activo y funcionando". Cuando se accede a la ruta /estado, el servidor responde con un objeto JSON que contiene información sobre el estado del servicio. Esto permite verificar que la API está funcionando correctamente y que puede responder solicitudes de los usuarios.
